Funshapes

Editor 2D graphical objects <C++/Qt5>

Download this project as a .zip file Download this project as a tar.gz file

Редакторнаписан на языке C++ с использованием кроссплатформы Qt5. Он может служить шаблоном для начинающих программистов использующих Qt для создания собственных программ использующих 2D графику. В коде программы можно найти примеры:

 

1.       Класса <QSignalMapper>как коммутатора событий/сигналов возникающих в программе

2.       Запись в файл, чтение из файла <QFile, QFileDialog, QTextStream>

3.       Создание собственного контексного меню <QMenu>

4.       Реализация механизма отмены/восстановления произведённых пользователем действий (Undo/Redo). Он же может быть использован для автоматизации действий программы - запись/воспроизведение макросов, анимация.

5.       Трансформации графического вида сцены/объектов (масштаб, вращение) <QTransform>

6.       Работа с видом сцены <QGraphicsView> преобразование координат (mapFrom…)

7.       Рисование с использованием <QPainterPath>

8.       Переопределение метода _paint() _объекта для управления процессом отрисовки (использование градиентной заливки, “подсветки” , определение порядка отрисовки пересекающихся объектов, управление масштабом рисования) <QPainter>

9.       Преобразование бинарных данных вBase64и обратно <QString,QByteArray>

10.   Выделение нескольких объектов мышью <Rubber Band Region>

 

Редакторпозволяет создавать и редактировать встроенные типы объектов, такие как:

1.          Треугольник

2.          Прямоугольник

3.          Круг

4.          Текст

5.          Рисунок

 

У** объектов** могут быть изменены следующие параметры:

1.          Положение

2.          Геометрия (для объектов “Рисунок”/”Текст” изображение/текст масштабируется под объект)

3.          Ориентация (угол наклона по оси Z)

4.          Цвет (кроме объекта “Рисунок”)

5.          Встроенный текст объекта а также шрифт, размер и цвет этого текста (кроме объекта “Текст”)

 

Сценаподдерживает работу с буфером обмена (Cut/Copy/Paste) и обладает механизмом отмены/восстановления изменений объектов (Undo/Redo). Состояние сцены с объектами и все их параметры (включая изображения объектов “Рисунок”) могут быть сохранены в файл. Файл сцены является простым текстовым файлом где сцена, объекты и их параметры описываются вXMLформате.
